# wox时间戳转换插件：TimeStampTranslate
#### 对 wox 不了解的朋友可以看下 [wox 官网](http://www.getwox.com/) ，个人感觉还是很好用的，键盘党福利，Windows 上的 Alfred。  
==**导入：具体的可看官方文档，整个项目下载下来。解压到指定插件文件夹即可。**==

---


本插件主要是解决时间戳的转换问题。具体如下：

在 wox 中 输入 “tt” 即可激活本插件
